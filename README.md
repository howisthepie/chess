# Chess

React + Vite шахматы против Stockfish с Supabase Auth, историей партий и настройками пользователя.

## Запуск

1. Установите зависимости:

```bash
npm install
```

2. Создайте `.env.local` на основе `.env.example`.
3. Примените SQL из `supabase/migrations/001_initial_schema.sql` в Supabase.
4. Запустите приложение:

```bash
npm run dev
```

## Проверка

```bash
npm test
npm run build
```
